# React DND
A simple drag and drop package, implemented using react hooks

## Install
`npm install react-dnd --save`
or
`yarn add react-dnd`

## Requirements

Must have react 16.8+ installed

## Getting started
1. run `yarn` to install react dependencies
2. run `yarn start` to start the project
